What do the scores mean?
  • Keywords — lexical match between your query and each scientist’s indexed text (publication titles/keywords or profile fields, depending on mode).
  • Semantic — embedding similarity: how close the scientist’s text is to the query in meaning, not just shared words.
  • Community — co-author network score for this query: we seed the graph with the top keyword matches, then spread rank along co-authorship links (more shared papers = stronger ties). High values mean the person is well connected to others who already match the topic.

Keywords, Semantic, and Community are raw scores (BM25, cosine similarity, graph PageRank). Profiles with no indexed text are excluded. Zero keyword score is allowed in the pool. Final uses min–max normalized signals over that pool and your fusion weights.
